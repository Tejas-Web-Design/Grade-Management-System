# Installation Guide

## Grade Management System

### System Requirements

Before running the project, ensure the following software is installed:

* Java Development Kit (JDK 8 or higher)
* Eclipse IDE / IntelliJ IDEA / VS Code
* Windows, Linux, or macOS

---

## Project Setup

### Step 1: Download the Project

Clone or download the project from GitHub.

```bash
git clone https://github.com/Tejas-Web-Design/Grade-Management-System
```



---

### Step 2: Open Project

Open your preferred IDE and import the project folder.

---

### Step 3: Navigate to Source Folder

Open terminal inside the src folder.

---

### Step 4: Compile the Project

```bash
javac *.java
```

This command compiles all Java source files.

---

### Step 5: Run the Application

```bash
java GradeManagementSystem
```

---

## Troubleshooting

### Java Not Recognized

Verify Java installation:

```bash
java -version
```

### Compilation Errors

Ensure all source files are present inside the src folder.

### IDE Issues

Refresh the project and rebuild the workspace.
