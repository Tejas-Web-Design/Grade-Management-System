# 📖 Project Overview

The **Grade Management System** is a Java-based console application designed to manage student academic performance efficiently. The system enables users to store student marks, calculate averages, assign grades automatically, identify top performers, and generate detailed performance reports.

This project demonstrates the practical implementation of:

* Object-Oriented Programming (OOP)
* Arrays and ArrayLists
* Exception Handling
* Input Validation
* Menu-Driven Programming
* Report Generation
* Academic Performance Analysis

# 🚀 Installation & Execution

### Clone Repository

```bash
git clone https://github.com/Tejas-Web-Design/Grade-Management-System/
```

### Navigate to Source Folder

```bash
cd src
```

### Compile

```bash
javac *.java
```

### Run

```bash
java GradeManagementSystem
```
